Chrome DevTools MCP Server 简介

DevTools MCP（MCP = Model Context Protocol）是一个用于与 Chrome DevTools 通信并远程调试或扩展 DevTools 的机制/服务器。工作坊会演示如何启动 MCP Server、连接 DevTools，以及如何使用 MCP 进行自定义调试体验。

关键点

  • 安装与启动 MCP Server（示例命令将在工作坊笔记中提供）
  • 如何在 DevTools 中通过 MCP 连接到目标（例如远程设备或模拟环境）
  • 常见调试场景与故障排查

快速示例（占位）

# 启动 MCP Server（示例）
# node mcp-server.js --port=9229
